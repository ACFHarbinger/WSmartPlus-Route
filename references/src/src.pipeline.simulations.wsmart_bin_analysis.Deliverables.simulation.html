class src.pipeline.simulations.wsmart_bin_analysis.Deliverables.simulation.Simulation(sim_type: str, ids: list, data_dir: str, train_split: str = None, start_date: str = None, end_date: str = None, rate_type: str = None, predictQ: bool = False, info_ver: str = None, names: str = None, savefit_name: str = None)#

Bases: src.pipeline.simulations.wsmart_bin_analysis.Deliverables.simulation.GridBase

A simulation class that extends GridBase to provide time-stepping and collection logic.

Initialization

Initialize the simulation with a specific type and time range.

Args: sim_type (str): Type of simulation (‘sampled’, ‘real’, or ‘real+sampled’). ids (list): List of bin IDs to simulate. data_dir (str): Root directory for data files. train_split (str, optional): Split date for training/testing. start_date (str, optional): Start date of the simulation. end_date (str, optional): End date of the simulation. rate_type (str, optional): ‘mean’ or ‘crude’. predictQ (bool): Whether to use a predictor. info_ver (str, optional): Info file version suffix. names (str, optional): Manual filenames. savefit_name (str, optional): Filename for saved model weights.

Advances the timeStep of the simulation by updating the fill level according to the rates the update procedure defined at init.

Parameters

date: str, datetime date to fetch values. string format %d-%m-%Y or datetime

Returns
