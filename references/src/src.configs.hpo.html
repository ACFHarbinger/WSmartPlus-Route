class src.configs.hpo.HPOConfig#

Hyperparameter optimization configuration.

Attributes: method: HPO method (‘dehbo’, ‘rs’, ‘gs’, ‘bo’). metric: Optimization metric (‘reward’, ‘cost’). n_trials: Number of HPO trials. n_epochs_per_trial: Training epochs per trial. num_workers: Number of parallel workers for HPO. search_space: Dictionary defining the search space.
