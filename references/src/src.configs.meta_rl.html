class src.configs.meta_rl.MetaRLConfig#

Meta-RL and HRL algorithm configuration.

Attributes: use_meta: Whether to use meta-learning wrapper. meta_strategy: Meta-learning strategy (‘rnn’, ‘bandit’, ‘morl’, etc.). meta_lr: Learning rate for meta-optimizer. meta_hidden_dim: Hidden dimension for meta-network. meta_history_length: History length for meta-learning. hrl_threshold: HRL threshold parameter.
