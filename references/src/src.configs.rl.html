class src.configs.rl.RLConfig#

RL algorithm configuration.

Attributes: algorithm: RL algorithm name (‘reinforce’, ‘ppo’, ‘sapo’, etc.). baseline: Baseline type (‘rollout’, ‘critic’, ‘pomo’, ‘warmup’). entropy_weight: Weight for entropy regularization. max_grad_norm: Maximum gradient norm for clipping.
